# Usage

## Basic Usage

```bash
favicon-generator generate logo.svg
```

## Advanced Example

```bash
favicon-generator generate logo.svg \
  --output-dir static/favicons \
  --webp \
  --avif \
  --optimize \
  --app-name "My Awesome App" \
  --theme-color "#2563eb" \
  --background-color "#ffffff"
```

## Command Options

```
favicon-generator generate --help
```
